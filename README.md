# dl-ppc64le

This is the repo for the Power8 docker image jayachandranb/deeplearning:ppc64le. The image requires nvidia-docker to run.

> nvidia-docker run -d -p 8888:8888 -p 7777:7777 --name=dl jayachandranb/deeplearning:ppc64le

The image has python 3.5.2 as default and the following python modules present/installed.
